# Simulatore di un lotto in troticoltura

## Requisiti

Per eseguire il progetto è necessario avere installato:

- Python 3
- la libreria esterna `rich`

## Installazione

È consigliato utilizzare un ambiente virtuale.

### 1. Creare l'ambiente virtuale
```bash
python -m venv .venv
```

### 2. Attivare l'ambiente virtuale

Su Windows:
```bash
.venv\Scripts\activate
```

Su macOS/Linux:
```bash
source .venv/bin/activate
```

### 3. Installare le dipendenze
```bash
pip install -r requirements.txt
```

## Esecuzione

Per avviare il simulatore:

```bash
python simulatore_troticoltura.py
```
